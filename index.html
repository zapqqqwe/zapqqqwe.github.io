面向长视频理解的 Agentic VideoLLM 框架设计

提出 VideoThinker，通过内置 Temporal Retrieval 与 Temporal Zoom 机制，将长视频理解从被动采样升级为工具驱动的主动式时序推理。

  • 提出 Retrieval + Zoom 的统一 Agentic 视频推理框架，实现主动时序定位与关键帧细粒度理解。
  • 利用高性能 LLM 自动生成 tool-use trajectory，构建视频帧交错的推理监督数据，用于 7B VideoLLM 训练。
  • 设计内置式 Agentic VideoLLM 架构，相比外置 Agent pipeline 降低模态转换损耗并提升推理稳定性。
  • 在 MLVU 与 LVBench 上取得 6.8% - 10.6% 的性能提升。
